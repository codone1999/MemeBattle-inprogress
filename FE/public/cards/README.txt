CARD IMAGES FOLDER

Place card artwork here with these naming conventions:

Examples:
- dragon-strike.png
- fireball.png
- ice-shield.png
- default-card.png (fallback for missing cards)

Specs:
- Format: PNG or JPG
- Size: 600x800px (3:4 ratio)
- Max file size: 300KB

To use in database:
{
  "cardImage": "/cards/dragon-strike.png"
}
