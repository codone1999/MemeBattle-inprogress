CHARACTER IMAGES FOLDER

Place character portraits here with these naming conventions:

Examples:
- shadow-tactician.png
- fire-mage.png
- ice-warrior.png
- default-character.png (fallback for missing characters)

Specs:
- Format: PNG (transparent background recommended)
- Size: 512x512px or 1024x1024px (square)
- Max file size: 500KB

To use in database:
{
  "characterPic": "/characters/shadow-tactician.png"
}
