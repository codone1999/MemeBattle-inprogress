## Overview

The MemeBattle Card Game Server is a real-time multiplayer card game backend built with Express.js, Socket.IO, MongoDB, and Redis.

### Key Technologies
- **Express.js** - REST API
- **Socket.IO** - Real-time communication
- **MongoDB** - Persistent storage
- **Redis** - Active game state
- **JWT** - Authentication (httpOnly cookies + refresh tokens)
